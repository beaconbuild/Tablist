Danke das du dieses Plugin heruntergeladen hast. Das erfreut mich sehr. 
Das Plugin ist für die Minecraft 1.15.2 gemacht. 
Installation:

Downloadet die Minecraft Spigot Version 1.15.2 von diesem Link: https://getbukkit.org/get/hJhavPDjhYFfqjyPAibsKAJecPpl7Z2V
Wenn ihr sie gedownloadet habt dann erstellt ein Ordner und in diesen zieht ihr die Spigot 1.15.2 Datei.
Ihr macht Doppelklick auf die Datei, dann öffnet sich ein Terminal. Dieser schlißt sich nach kurzer Zeit und es erstellt sich eine Eula.txt. 
Auf diese Klickt ihr und musst false auf true stellen und die datei speichern. 
nun öffnet ihr nochmal die Datei und es erstellen sich ordner.
Ein Ordner davon ist Plugins. 
In diesen Ordner zieht ihr die Plugin Datei.
Dann erstellt ihr eine TXT datei die ihr start.txt nennt.
In diese Datei kopiert ihr diesen Code:  java -Xmx1G -jar SERVER_FILE_NAME.jar
PAUSE

SERVER_FILE_NAME ersetzt ihr durch den namen eurer Server datei.
Dann Speichert ihr die txt datei als Start.bat. Ganz wichtig ist das dahinter .bat steht.

Wenn ihr nun den Server starten wollt drückt auf die Start.bat datei.

Dann startet der Server und das Plugin funktioniert :D
